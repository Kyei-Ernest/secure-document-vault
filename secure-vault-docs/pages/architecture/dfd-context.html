Context Level DFD (Level 0)

The Context Level Data Flow Diagram provides the highest-level view of data movement through the Secure Document Vault. It shows the system as a single process, with data flows to and from external entities including users, administrators, external KMS, and audit systems.

Diagram

Context Level DFD (Level 0)

External Entities

  • End Users — submit documents, retrieve encrypted files, manage permissions
  • Administrators — configure system settings, manage users and roles
  • Key Management Service — provides master keys for envelope encryption
  • Audit System — receives immutable log entries for compliance
  • External Auth — MFA verification through TOTP/FIDO2 providers

Key Data Flows

  • Document upload → encryption → encrypted storage
  • Authentication request → credential verification → JWT issuance
  • Document retrieval → authorization check → decryption → delivery
  • All operations → audit log generation → immutable storage
