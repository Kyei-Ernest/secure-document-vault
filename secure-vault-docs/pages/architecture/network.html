Network Architecture

The network architecture implements strictly defined security zones to control traffic flow and isolate sensitive components. It adheres to the principle of least privilege at the network layer.

Diagram

Network Architecture - Security Zones & Firewall Rules

Security Zones

Zone Access Components
Public Zone Internet facing Load Balancers, WAF, NAT Gateways
DMZ Restricted inbound Bastion Hosts, VPN Terminators
App Zone Internal only Microservices (Auth, Doc, Audit, Encrypt)
Data Zone Strictly isolated PostgreSQL, Redis, KMS Interface

Firewall Rules

ℹ️ Default Deny

All network traffic is denied by default. Allow rules are explicitly defined for required communication paths only.

  • Ingress: HTTPS (443) only from WAF to Load Balancers.
  • App-to-DB: Port 5432 allowed only from App Zone to Data Zone.
  • Egress: Restricted to specific FQDNs (e.g., KMS endpoints, SMTP) via Egress Filter.
