Level 1 Data Flow Diagram

The Level 1 DFD decomposes the single system process from the Context Level into four major subsystems: Authentication & Authorization, Document Management, Encryption/Decryption, and Audit & Logging. Each subsystem shows its data flows to internal data stores and external entities.

Diagram

Level 1 DFD — Major Process Decomposition

Major Processes

ProcessIDDescription
Authentication & Authorization1.0User identity verification, JWT management, RBAC/ABAC enforcement
Document Management2.0Document CRUD, versioning, search, sharing workflows
Encryption/Decryption3.0AES-256-GCM encryption, key management, envelope encryption
Audit & Logging4.0Immutable event logging, hash chaining, compliance reporting

Data Stores

  • D1: User Database — credentials, roles, MFA secrets
  • D2: Document Database — metadata, versions, permissions
  • D3: Encrypted Storage — encrypted document blobs
  • D4: Key Store — encrypted DEKs, key versions
  • D5: Audit Log — immutable, hash-chained event records
